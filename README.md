# Factory5
Factory5 task

Функция choose_elements - ответ на первый вопрос.
Сложность алгоритма линейная, при k < n/2 - O(k), при k > n/2 - O(n). Случай k —> N проблем не вызывает.

Функция choose_elements_prob - ответ на второй вопрос.
Сложность алгоритма линейная, при k < n/2 - O(k), при k > n/2 - O(n). Случай k —> N проблем не вызывает.
